# i3 Window Manager

## Change Window

    MOD + hjkl

## Move window to `<direction>

    MOD + HJKL

## Move window to external

    MOD + x

## Move window to internal

    MOD + X

## Layout stacked

    MOD + s

## Layout tabbed

    MOD + t

## Toggle split direction

    MOD + z

## Toggle split orientation

    MOD + e

## Toggle fullscreen

    MOD + f

## Reload/restart i3

    MOD + C/R

## Resize

    MOD + Ctrl + hjkl

## Next/prev workspace

    MOD + o/i
